# Release Notes — 2026-03-27

## Version
v1.2.1

## Scope
House pages, grouped stays, route consistency, recommendation data, and hero selection cleanup.

## Included changes
- Fixed grouped-stay card behavior to avoid misleading single-house destination behavior.
- Corrected grouped-stay metadata so filters reflect defensible group-level truth.
- Corrected villa route linking on the location page.
- Added recommendation-only suggested pairing data for nearby house combinations.
- Added member links inside suggested pairing blocks on house pages.
- Cleaned up featured-image selection so hero choice is more deterministic where needed.
- Restored/fixed Kalliopi hero image path/selection behavior.
- Removed unused legacy message utility.

## Notes
- Suggested pairings are recommendation-only and are separate from official grouped inventory.
- Official grouped stays remain the only filterable grouped inventory objects.
- Agent/workflow files were intentionally excluded from this release.
