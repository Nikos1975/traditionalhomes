# Purpose

Use this skill whenever writing or revising public-facing copy for the website.

# Brand voice rules

- Calm, precise, understated
- No hype, urgency, or sales language
- No clichés or generic luxury phrasing
- Use concrete details over adjectives
- Prefer trust, clarity, and factual accuracy
- Remove unsupported claims
- Show, do not sell

# Workflow

1. Identify content type:
   - house page
   - villa page
   - blog post
   - guide page
   - general site copy

2. Audit the draft for:
   - hype or promotional tone
   - vague descriptors
   - generic travel language
   - repetition
   - unsupported claims
   - missing practical information
   - weak information hierarchy

3. Rewrite according to content type:

## House / villa page
- brief factual overview
- layout and flow
- sleeping arrangement
- outdoor space / view / pool
- who it suits
- practical constraints or access notes

## Blog / guide page
- grounded opening scene
- geography / setting
- specific observations
- useful practical relevance
- quiet closing without promotion

## General site copy
- clear purpose
- factual explanation
- concrete details
- concise next step if needed

4. Output format
- revised copy
- key edits made
- unsupported claims removed
- factual gaps still needing confirmation

# Constraints

- Do not invent facts
- If a fact is uncertain, mark it as needing confirmation
- Preserve structure when possible
- Prefer minimal necessary rewriting over stylistic over-editing
